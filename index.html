Experiments

  • 2020.02.23
    Face - edit both the text and the font it is rendered in.
  • 2019.01.08
    Pal - apply an eight-color terminal color scheme to an image. Use the keyboard controls to choose a theme, set thresholds, and cycle hues.
  • 2019.12.28
    Bix - draw on binary to glitch text.
  • 2019.11.08
    Diptych - pixel-reflow an image to match the dimensions of your text. Save the result as a diptych.
  • 2019.10.29
    Slide - divide and slide-stretch an image using keyboard controls.
  • 2019.10.08
    Freeconfig - push around image pixels in blocks.
  • 2019.10.04
    Moire - generate angular skyscapes using Asteroids-like ship controls.
  • 2019.09.25
    Hex - a keyboard-driven, grid-based drawing tool.
  • 2019.09.19
    Etch - a keyboard-based pixel drawing tool.
